# Hi, I'm Freddy 👋

I build automation that keeps **healthcare communications running** — at a scale of 40,000+ endpoints, 15–18 million contact center calls per year, across four U.S. states.

Most of my work lives at the intersection of **Cisco Unified Communications**, **Python automation**, and **regulatory compliance** (E911 / RAY BAUM's Act / Kari's Law). I write code that replaces weeks of manual engineering work with scripts that run in minutes.

I open-source the tools and methodologies I build so other UC engineers in healthcare don't have to start from scratch.

---

**What I work on**

- 🏥 **Healthcare UC Infrastructure** — CUCM, Unity Connection, UCCX/UCCE, Expressway, CVP, CER, IM&P, Jabber, Webex across large multi-state health systems
- 🐍 **Python + AXL Automation** — Bulk provisioning, phone/gateway discovery, firmware compliance auditing, certificate lifecycle management
- 📞 **E911 Compliance** — Automating RAY BAUM's Act and Kari's Law implementation for healthcare multi-line telephone systems
- 🔄 **CUCM Upgrade Toolkits** — Automated hardware/firmware/IOS compatibility validation against Cisco's live v15 Compatibility Matrix
- 🧪 **Contact Center Engineering** — UCCE/UCCX/CVP scripting, IVR optimization, Calabrio WFM integration
